How to install the Bambora Checkout

Install on your server:

  1. Download the code on your server.
  2. Also add the module folder path to your Odoo Conf file.
  3. Restart your server.
  4. Go to Settings. Click on "Activate the developer mode".
  5. Go to Apps. Click on "Update Apps List" and then click "Update".
  6. Remove all the filters and search by "Bambora Checkout".
  7. Click on Install.

For Production

Bambora Checkout Configurations

You will need the merchant ID, payment API and Profile API from the Bambora.

1. Go to Administration --> Account Settings --> Order Settings. Note down the: API access passcode

2. Go to Configuration --> Payment Profile Configuration. Note down the: API access passcode

Odoo Configurations for Bambora Checkout

Put the merchant ID, payment API and Profile API from the Bambora

Select "Payment from Odoo" as Payment Flow and "Bank" as Payment Journal.

Card Payment using Bambora Checkout

Select Bambora Checkout in Website

Add card details and click on Pay

Payment successful message will be shown.
