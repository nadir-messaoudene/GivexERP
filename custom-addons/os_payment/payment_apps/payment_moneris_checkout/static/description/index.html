How to install Moneris Checkout

Installation on the Server

  • Download code on your server.
  • Add the module folder path to your Odoo confg file.
  • Restart the server.
  • Go to Settings and click on "Activate the Developer Mode".
  • Go to Apps and click on "Update Apps List". Then, click on "Update".
  • Remove all the filters and search by "payment_moneris_checkout".
  • Click on Install.

How to Login to Moneris Site

  • Login to Moneris site www3.moneris.com/mpg  
  • Use your Username, Store ID and Password to login to Moneris site.

How to Install for Testing

  • Go to Moneris testing URL https://esqa.moneris.com/mpg
  • Login with Username: demouser, Store: store3 and Password: password.
  • For creating a test store, please look at the end of this page: Steps to create a Moneris Test Account.

How To Configuration

Moneris Panel

  • Login to Moneris website
  • Go to “Admin” and then “Moneris Checkout Confg”
  • Click on "Create Profile"
  • Your profile will be created
  • Take a note of the “Checkout ID” and review configurations as required
  • Select "I have my custom order form and want to use Moneris simply for payment processing".
  • Select your preferred currency.
  • Go to Payment >Transaction Type, Select "Purchase".
  • Go to Payment > Gift Cards and select "Accept Gift Cards".
  • Under Order Confirmation select "Use Moneris".

Moneris Checkout in Odoo

  • Go to Invoicing > Payments > Payment Acquirers > Moneris Checkout in Odoo.
  • Add the Store ID and API Token related to tour Moneris Store. Also Publish it for showing on the website. Note: This information can be found on the Moneris platform.
  • Your profile will be created
  • Select 'Bank' as Payment Journal and 'Payment from Odoo' as Payment Flow. Saving Card Data must be enabled.

How To Create A Test Account

Steps to create Moneris test account

  • Go to https://developer.moneris.com/
  • Add all the required information
  • Check you email for verification
  • Click on the verify URL
  • Click on My Profile icon
  • Enter your credentials to login
  • Click on “Full Profile” button
  • Click on “Request Testing Credentials”
  • Enter your password and select country “Canada” and click on “REQUEST TEST ACCOUNT”
  • Your credentials will be shown. Take a not of the store id and API token. Use them in the Odoo configuration.
