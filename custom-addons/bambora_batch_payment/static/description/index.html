Integration of Bambora EFT/ACH with Odoo eCommerce and Sales

Bambora, one of the largest financial technology companies in Canada, specializes in payment processing. With this app, you can make transactions - Bank to Bank - easily and securely.

Bambora EFT is a function that allows you to pay Bills, Invoices instantaneously and collect customer payments through Banks from Odoo websites and internal/portal sites. Also, you can make multiple payments batch-wise.

Screenshots

1. Vendor Bill Payment using Bambora EFT

Must set vendor accounts information in vendor profile → invoicing

Create a Bill and others tab must select a bank account which must have all bank information.

Select as many bill want to pay through EFT . But must follows flowing criteria :
  1. Must have bank account with Account Number,Identification Number(3 digit),Institution Number(5 digit)
  2. Bill must be in posted State
  3. Not paid previously or in Ecommerce transaction state

Click ‘Bambora Batch Payment’ from action for initiate Bambora EFT request.

After Successful Request in Bambora Status Column we can see the Current state . It might take ¾ days to complete the transaction and every step can seen from this column.

All Bambora EFT requests and details can be seen from Vendors → Bambora Batch Payments.

EFT details can be seen from each Invoice/Bill in others bamora batch payments.

If Bambora Status is ‘Complete’ then Bill/Invoice will be auto paid.

N.B - Same Procedure for Customer Invoice from Backend .

Installation Guideline

Install on your server:

  1. Download the code on your server.
  2. Also add the module folder path to your Odoo Conf file.
  3. Restart your server.
  4. Go to Settings. Click on "Activate the developer mode".
  5. Go to Apps. Click on "Update Apps List" and then click "Update".
  6. Remove all the filters and search by "Bambora EFT/ACH".
  7. Click on Install.

For Production

Bambora Merchant Online https://merchant.bambora.com/auth/signin/ using your Username and Password.

For Testing

Go to Bambora Testing Url: https://web.na.bambora.com/admin/sDefault.asp.
Login with Company login, User login and Password.

For creating a test Account, Fill out our sign up form to instantly get a developer account with access to a sandbox..

Bambora Bank-to-Bank EFT Configurations

Go to Invoicing -> Configuration -> Payment Acquirers

Select Bambora EFT

Select State, Company and Website

Put the merchant ID, payment API and Profile API from the Bambora under Bambora Credentials

Put the Batch API, Report API,Transaction Type(EFT for Canada and ACH for USA). Activate 'Create Profile' if you want to tokenize Bank using Bambora.

Activate 'Create Profile' if you want to tokenize Bank using Bambora.

Go to Configuration Tab.
  1. Select Payment from Odoo as Payment Flow
  2. Select Payment Journal for Customer Payments
  3. Select Vendor Payment Journal for Vendor Payments if you want to activate Vendor Payments

Go to Messages Tab. You can edit messages according to your needs.
