Journal Entry Items Import from Excel

Import Journal Entry / Items Odoo Module

This app allows your accounting team to import journal items from excel files.

Main Features:

  • Allow you to import journal items from excel file into Odoo.
  • Excel samples are available inside the module.
  • Make sure you have xlrd Python module installed. https://pypi.python.org/pypi/xlrd
  • For more details please see video and below screenshots.

Following fields can be import:

  • ACCOUNT CODE
  • PARTNER
  • LABEL
  • AMOUNT CURRENCY
  • CURRENCY
  • DEBIT
  • CREDIT
  • DUE DATE
  • REFERENCE
  • DATE
